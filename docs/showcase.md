# Showcase And Use Cases

This page is the public showcase layer for US Video API.

It is intentionally focused on practical use cases that developers, agencies, and enterprise evaluators can understand quickly.

## What US Video API Is Good For

- short-form ad generation
- ecommerce product launches
- local business promotional creatives
- creative automation inside apps and internal tools
- AI agent workflows that need video generation as a callable capability
